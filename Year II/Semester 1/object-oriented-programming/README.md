# Object-Oriented Programming
*Programare orientată pe obiect (POO) — Year II, Semester 1*

Object-oriented programming in C++: classes and objects, encapsulation, inheritance, polymorphism, operator overloading and templates.

## Materials
- **Lectures/** — course notes (2022).
- **Colloquia/** — 11 lab practical tests (*colocvii*), 2014–2023.
- **Exams/** — 32 files: written exams 2005–2014 and a `Barem examen POO - 14 iunie 2021/` answer key.
- **Project 1/**, **Project 2/**, **Project 3/** — the three graded C++ projects (source `.cpp`, Code::Blocks `.cbp` and the assignment briefs).
