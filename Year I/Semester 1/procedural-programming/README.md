# Procedural Programming
*Programare procedurală (PP) — Year I, Semester 1*

First programming course, taught in C: data types, control flow, functions, arrays, pointers and strings.

## Materials
- **Exam session/** — 59 photos/screenshots of solutions from the practical exam (two parts).
- **Subiecte examen ani anteriori - Documente Google.pdf** — past-year exam topics.
- **Probleme laboratoare seria 10.docx** — lab problem set (series 10).
- **Examen PP.docx**, **Bonus PP.pdf** — exam-prep material.
- **Cheat.txt** — a C syntax cheat sheet.
- **Document.rtf** — miscellaneous notes.
