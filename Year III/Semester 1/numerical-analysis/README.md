# Numerical Analysis
*Analiză numerică și metode numerice — Year III, Semester 1*

Numerical methods: polynomial interpolation (Lagrange, Newton, Neville), splines, numerical differentiation and integration (rectangle, trapezoid, Simpson).

## Materials
- **Labs/** — 23 files in `Exercitii laborator/`: lab exercises and an `Algoritmi/` (Algorithms) subfolder of Python implementations (Lagrange, Newton divided differences, Neville, linear/quadratic/cubic splines, quadrature rules, finite differences), plus the homework scripts.
- **Exams/** — 22 files: exams 2018–2024, several with worked solutions (*rezolvat*).
