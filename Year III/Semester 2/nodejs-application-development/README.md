# NodeJS Application Development
*Programarea aplicațiilor în NodeJS — Year III, Semester 2*

Building back-end web applications with Node.js: Express routing, controllers/middleware and a Prisma ORM data layer.

- **Projects/** — the graded REST API: `app.js`, `client.js`, a Prisma schema (`prisma/`) and a structured `src/` (controllers, routes, middlewares).
- **Cerinte proiect.pdf** — project requirements.
